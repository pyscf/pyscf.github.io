PySCF Virtual Seminar Series#

The PySCF Virtual Seminar Series is a monthly forum where PySCF developers can present, share, and advertise new or ongoing implementations across the PySCF software suite. The seminar is open to everyone, join live via Zoom meeting (there won’t be any recording of the talk), and if you’d like calendar invites so you don’t miss a talk, please fill out the Google Form. If you’re interested in presenting in a future session, you can submit the form.

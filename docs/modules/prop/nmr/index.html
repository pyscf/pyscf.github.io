    prop.nmr — NMR shielding tensor¶

    The prop.nmr module implements the nuclear magnetic resonance (NMR) shielding tensor for mean-field methods.

    Examples¶

    • source

      examples/prop/04-nmr.py

    get_fock(dm0=None, gauge_orig=None)¶

    First order partial derivatives of Fock matrix wrt external magnetic field. frac{partial F}{partial B}

    kernel(mo1=None)[source]¶

    Kernel function is the main driver of a method. Every method should define the kernel function as the entry of the calculation. Note the return value of kernel function is not strictly defined. It can be anything related to the method (such as the energy, the wave-function, the DFT mesh grids etc.).
